# 🚀 Overview

This project is a console-based **Personal Finance Tracker System** developed using **Python** with **CSV File Handling**.

The system helps users manage their financial activities such as expenses, debts, budgets, and savings through a simple menu-driven interface.

The project uses only the following Python libraries:

- `csv`
- `datetime`
- `matplotlib`

CSV files are used as a lightweight database system for storing and managing financial records.
